# LatihSoal

Aplikasi web latihan soal mandiri dan kolaboratif berbasis React, Vite, dan TypeScript.

## Jalankan Lokal

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy ke Vercel

1. Import repository ini di Vercel.
2. Gunakan preset `Vite`.
3. Build command: `npm run build`
4. Output directory: `dist`
